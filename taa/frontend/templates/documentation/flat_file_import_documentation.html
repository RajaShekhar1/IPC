5Star Flat File and FTP Import Documentation

5StarEnroll supports importing enrollment data via a flat-file import service. Authorized 3rd party integrators can upload correctly formatted files to our secure FTP drop box, and 5StarEnroll will process the file once it passes a validation check.

Please contact us at support@5starenroll.com if you have any questions.

Flat File Specification

The 5StarEnroll system accepts a flat-file enrollment format consisting of UTF-8 encoded text characters. The content of the file is:

  • One line of header fields
  • One line of record fields for each enrollment record being transmitted

Each field has a particular size and data type restrictions given in the tables below. If a given data value is smaller than the prescribed size, the field must be padded with whitespace characters. The size given is a character count.
